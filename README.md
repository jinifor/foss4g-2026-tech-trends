# FOSS4G 2026 Tech Trends

FOSS4G 2026 발표 목록을 키워드, 오픈소스 라이브러리, 클라우드, AI 관점에서 읽기 쉽게 재구성한 정적 분석 대시보드입니다.  
React 기반 인터랙티브 시각화로 발표 주제와 기술 흐름을 빠르게 탐색할 수 있습니다.

## 프로젝트 개요

이 저장소는 단순한 발표 목록을 넘어서, FOSS4G 2026 발표 제안서 집합에서 어떤 기술과 주제가 많이 등장하는지 빠르게 파악할 수 있도록 만드는 것을 목표로 합니다.

- `Keyword Insight`: 전체 발표에서 추출한 공간정보/기술 키워드 분석
- `Library Insight`: 명시적으로 언급된 오픈소스 라이브러리와 도구 분석
- `Cloud Insight`: 클라우드 인프라, 저장소, 오케스트레이션, 배포 관련 신호 분석
- `AI Insight`: AI 키워드, AI 패밀리, 맥락 키워드 간 관계 분석
- 다국어 지원: `en`, `ko`, `ja`

이 앱은 백엔드 없이 동작하는 Vite 기반 SPA입니다.

## 현재 분석 범위

현재 앱에서 확인할 수 있는 분석 범위 요약입니다.

- 전체 발표 수: `376`
- 키워드 대시보드: 고유 키워드 `207`, 총 언급 수 `765`
- 라이브러리 대시보드: 고유 라이브러리/도구 `50`, 총 언급 수 `211`
- 클라우드 대시보드: 클라우드 관련 발표 `46`, 고유 신호 `22`, 총 언급 수 `77`
- AI 대시보드: AI 관련 발표 `56`, AI 키워드 `18`, 맥락 키워드 `53`

## 주요 기능

- 카드, 워드클라우드, 상위 키워드 차트, 버블 차트, 트리맵 제공
- 키워드/라이브러리/클라우드 신호의 공출현 네트워크와 히트맵 제공
- 클러스터 기반 요약으로 주제군을 빠르게 파악 가능
- 발표 제목, 초록, 키워드 기준 탐색 가능한 Explorer 제공
- 언어 선택 상태를 브라우저 `localStorage`에 저장

## 기술 스택

- 프론트엔드: React 19, TypeScript, Vite
- 스타일/UI: Tailwind CSS, Lucide React
- 시각화: Recharts
- 다국어: i18next, react-i18next

## 시작하기

### 요구 사항

- Node.js 및 npm
- Python 3

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

### 프로덕션 빌드

```bash
npm run build
```

### 빌드 결과 미리보기

```bash
npm run preview
```

## 사용 가능한 명령

- `npm run dev`: 개발 서버 실행
- `npm run build`: 타입 체크 및 정적 빌드
- `npm run preview`: 빌드 결과 로컬 미리보기

앱 자체는 정적 호스팅에 적합하며, 배포 산출물은 `dist/`에 생성됩니다.

## 이 프로젝트가 유용한 경우

- FOSS4G 2026 발표 목록을 빠르게 훑고 기술 트렌드를 파악하고 싶을 때
- 어떤 오픈소스 지리공간 라이브러리와 플랫폼이 많이 언급되는지 보고 싶을 때
- 클라우드/AI 관련 발표 비중과 세부 맥락을 따로 분석하고 싶을 때
- 발표 목록을 시각적으로 요약한 자료나 내부 공유용 분석 대시보드가 필요할 때
